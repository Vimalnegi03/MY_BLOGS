***HTML and HTML 5*** are different but many good developers too still think that they are the same. So let us understand how they are different HTML is Hyper Text Markup Language which is used to create web pages and HTML 5 is nothing but the latest version of HTML. **HTML 5** supports multimedia for this it has new tags like `audio` and `video.` It provides better <mark> SEO(Search Engine Optimization)</mark> which is essential in a world full of competition. There are new tags like `header footer` `nav` that increases the readability of code In HTML we just used to have `div` and `span` that used to make code very complex and modifications were not easy.

**HTML 5** developers are still working on it and there are chances that the feature that is accessible today may not be available tomorrow. **HTML 5** is used to create more <mark>interactive websites</mark> but if you want to create a *static website* then go for **HTML** as it is *fast*.
