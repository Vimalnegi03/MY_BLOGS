Fonts are very useful in increasing user experience on a web page. They make our web page look more attractive...Fonts present by default are not that much attractive so we can solve this problem by using google Fonts...Google provides more than 1500 fonts in this blog I will tell you how you can import these fonts on your webpage...

so steps to include the font in our web page are-:

1 Go to google and search google fonts.

2 Open the website and select the font you want to import

3 Copy the CDN of the font you want to import and paste it into the head of your HTML file.

4 Paste the font name in your CSS file and that it your fonts are ready to use.
