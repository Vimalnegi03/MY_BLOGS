HTML or *hypertext markup language* looks simple but it's the most essential requirement thing for creating a web page ...In simple words, we can say that if a web page is like a house then HTML is like the bricks that are needed to create that web page. A beginner thinks that HTML is just simple and many times ignores its importance but every successful web developer knows its importance...

HTML is so powerful that you can create a web page just by using **HTML**.

As a beginner, one should try creating projects only using HTML to have a good hold on this language and to understand the basics of a web page.
