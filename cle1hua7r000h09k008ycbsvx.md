*Emmet* is a **plugin** for *text editors* that allows us to write <mark>HTML and CSS faster. </mark> The biggest feature is that emment provides a boilerplate that consists of **meta tags, version of HTML and head and body tags** it provides *formatting* to our code mean by pressing `! + tab` keys you just get the complete structure of the page. It also provides a *closing* tag as soon as we *open* a tag hence it reduces the chances of getting errors in code. It also provides shortcuts to write <mark>CSS and HTML tags.</mark>

It is a tool kit for webdevlopers. The best part of emmet is that it's an application of **MICROSOFT** hence present by default in our favorite **vs code...**
